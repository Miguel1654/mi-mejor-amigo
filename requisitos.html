Requisitos

"No compres, ¡Adopta!"

Para comprar un perro, basta con tener dinero... Para adoptar un animal, basta con tener corazón.

  • Querer y respetar a los animales
  • Ser mayor de edad
    (Si eres menor, acudir con tus padres)
  • Tener espacio adecuado para tenerlo
Nota: Debe de cumplir el 100% de los requisitos para poder adoptar
